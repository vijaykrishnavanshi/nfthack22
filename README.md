NFTpay

###

Your own hosted invoices - represented as NFT's.

No fees.

### Running the project

NFTpay requires the following environment variables.

<pre>
REACT_APP_NFT_KEY={YOUR_NFT_STORAGE_KEY}
REACT_APP_NFT_KEY={YOUR_NFT_STORAGE_KEY}
</pre>

`yarn && yarn start`
